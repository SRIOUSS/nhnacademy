## 학습자료

- 산술연산자
    
    **기본 산술 연산자:**
    
    | 연산자 | 의미 | 예시 | 결과 |
    | --- | --- | --- | --- |
    | `+` | 덧셈 | `5 + 3` | `8` |
    | `-` | 뺄셈 | `5 - 3` | `2` |
    | `*` | 곱셈 | `5 * 3` | `15` |
    | `/` | 나눗셈 | `5 / 3` | `1` |
    | `%` | 나머지 | `5 % 3` | `2` |
    
    **정수 나눗셈 주의:**
    
    ```java
    int a = 5 / 3;      // 결과: 1 (소수점 버림)
    int b = 10 / 4;     // 결과: 2 (소수점 버림)
    
    // 소수점 결과가 필요하면 double 사용
    double c = 5.0 / 3;          // 결과: 1.666...
    double d = (double) 5 / 3;   // 결과: 1.666... (형변환)
    ```
    
    **나머지 연산자 활용:**
    
    ```java
    // 짝수/홀수 판별
    int num = 7;
    if (num % 2 == 0) {
    		System.out.println("짝수");
    } else {
    		System.out.println("홀수");
    }
    
    // 배수 판별
    if (num % 3 == 0) {
    		System.out.println("3의 배수");
    }
    ```
    
    **복합 대입 연산자:**
    
    | 연산자 | 의미 | 예시 | 동일한 표현 |
    | --- | --- | --- | --- |
    | `+=` | 더하고 대입 | `a += 3` | `a = a + 3` |
    | `-=` | 빼고 대입 | `a -= 3` | `a = a - 3` |
    | `*=` | 곱하고 대입 | `a *= 3` | `a = a * 3` |
    | `/=` | 나누고 대입 | `a /= 3` | `a = a / 3` |
    | `%=` | 나머지 대입 | `a %= 3` | `a = a % 3` |
    
    ```java
    int count = 10;
    count += 5;      // count = 15
    count -= 3;      // count = 12
    count *= 2;      // count = 24
    ```
    
    **증감 연산자:**
    
    | 연산자 | 위치 | 의미 |
    | --- | --- | --- |
    | `++` | 앞 | 먼저 증가, 후에 사용 |
    | `++` | 뒤 | 먼저 사용, 후에 증가 |
    | `--` | 앞 | 먼저 감소, 후에 사용 |
    | `--` | 뒤 | 먼저 사용, 후에 감소 |
    
    ```java
    int i = 5;
    System.out.println(i++);   // 출력: 5, i는 6이 됨
    System.out.println(++i);   // 출력: 7, i는 7이 됨
    System.out.println(i--);   // 출력: 7, i는 6이 뒴
    System.out.println(--i);   // 출력: 5, i는 5가 됨
    ```
    
- 연산 결과 타입
    
    **정수 연산 결과:**
    
    ```java
    int a = 5;
    int b = 3;
    
    int result1 = a + b;    // int + int = int
    int result2 = a / b;    // int / int = int (1, 소수점 버림)
    ```
    
    **정수와 실수 혼합 연산:**
    
    ```java
    int a = 5;
    double b = 2.0;
    
    double result = a / b;   // int / double = double (2.5)
    // 더 큰 타입으로 자동 변환 (int -> double)
    ```
    
    **타입 변환 규칙 (자동 승격):**
    
    | 연산 | 결과 타입 | 예시 |
    | --- | --- | --- |
    | byte + byte | int | byte, short는 int로 승격 |
    | short + short | int | byte, short는 int로 승격 |
    | int + int | int | `5 + 3` → `8` |
    | float + float | float | `1.5f + 2.5f` → `4.0f` |
    | int + double | double | `5 + 3.0` → `8.0` |
    | int / int | int | `5 / 3` → `1` |
    | int / double | double | `5 / 3.0` → `1.666` |
    | double / int | double | `5.0 / 3` → `1.666` |
    
    참고: Java에서 `byte`, `short`, `char` 타입은 산술 연산 시 자동으로 `int`로 승격됨
    
    **명시적 형변환(캐스팅):**
    
    ```java
    int a = 5;
    int b = 3;
    
    // 정수 나눗셈 -> 소수점 버림
    int result1 = a / b;           // 1
    
    // 형변환으로 실수 나눗셈
    double result2 = (double) a / b;   // 1.666...
    double result3 = a / (double) b;   // 1.666...
    double result4 = (double) (a / b); // 1.0 (이미 1로 계산된 후 변환)
    ```
    
    **문자열 연결 연산:**
    
    ```java
    String name = "Jave";
    int version = 21;
    
    // + 연산자로 문자열 연결
    String result = name + version; // "Java21"
    String result2 = version + name  // "21Java"
    
    // 주의: 연산 순서
    System.out.println(1 + 2 + '점')   // "3점" (1 + 2 = 3, 3 + "점")
    System.out.println("점수: " + 1 + 2);  // "점수: 12" (문자열 + 1 = "점수: 1", + "2")
    System.out.println("점수: " + (1 + 2));  // "점수: 3" (괄호 먼저)
    ```
    
- 비교 연산자
    
    **비교 연산자의 종류:**
    
    | 연산자 | 의미 | 예시 | 결과 |
    | --- | --- | --- | --- |
    | `==` | 같다 | `5 == 5` | `true` |
    | `!=` | 같지 않다 | `5 != 3` | `true` |
    | `>` | 크다 | `5 > 3` | `true` |
    | `<` | 작다 | `5 < 3` | `false` |
    | `>=` | 크거나 같다 | `5 >= 5` | `true` |
    | `<=` | 작거나 같다 | `5 <= 3` | `false` |
    
    **비교 연산자 활용:**
    
    ```java
    int age = 20;
    int limit = 18;
    
    boolean isAdult = age >= limit;  // true
    boolean isExact = age == limit;  // false
    boolean isUnder = age < limit;   // false
    ```
    
    **문자열 비교 주의:**
    
    ```java
    String a = "hello";
    String b = "hello";
    String c = new String("hello");
    
    // == 는 참조(주소) 비교
    System.out.println(a == b);   // true (같은 리터럴 풀)
    System.out.println(a == c);   // false (다른 객체)
    
    // equals()는 값 비교
    System.out.println(a.equals(b));  // true
    System.out.println(a.equals(c));  // true
    
    // 권장: null-safe 비교
    System.out.println("hello".equals(a));    // true
    ```
    
- 논리 연산자
    
    **논리 연산자 종류:**
    
    | 연산자 | 의미 | 설명 |
    | --- | --- | --- |
    | `&&` | AND | 둘 다 true여야 true |
    | `||` | OR | 하나라도 true면 true |
    | `!` | NOT | true↔false 반전 |
    
    **AND (&&) 연산:**
    
    ```java
    int age = 25;
    boolean hasLicense = true;
    
    // 둘 다 만족해야 true
    if (age >= 18 && hasLicense) {
    		System.out.println("운전 가능");
    }
    ```
    
    | A | B | A && B |
    | --- | --- | --- |
    | true | true | true |
    | true | false | false |
    | false | true | false |
    | false | false | false |
    
    **OR (||) 연산:**
    
    ```java
    String input = "quit";
    
    // 하나라도 만족하면 true
    if (input.equals("q") || input.equals("quit") || input.equals("exit") {
    		System.out.println("프로그램 종료")
    }
    ```
    
    | A | B | A || B |
    | --- | --- | --- |
    | true | true | true |
    | true | false | true |
    | false | true | true |
    | false | false | false |
    
    **NOT (!) 연산:**
    
    ```java
    boolean isFinished = false;
    
    if (!isFinished) {    // isFinished가 false이면 true    
    		System.out.println("아직 진행 중");
    }
    ```
    
    | A | !A |
    | --- | --- |
    | true | false |
    | false | true |
    
    **복합 조건:**
    
    ```java
    int score = 85;
    int age = 20;
    
    // AND와 OR 조합 (AND가 OR보다 우선순위 높음)
    if (score >= 80 && age >= 18 || score >= 90); {
    		System.out.println("합격");
    }
    
    // 괄호로 우선순위 명확히
    if ((score >= 80 && age >= 18) || score >= 90) {
    		System.out.println("합격");
    }
    ```
    
    **단락 평가 (Short-circuit Evaluation):**
    
    ```java
    // && : 앞이 false면 뒤는 평가 안 함
    if (false && someMethod()) { // someMethod() 실행 안 됨
    		// ...
    }
    
    // || : 앞이 true면 뒤는 평가 안 함
    if (true || someMethod() {   // someMethod() 실행 안 됨
    		// ...
    }
    
    // null 체크에 유용
    String str = null;
    if (str != null && str.length() > 0) {   // str이 null이면 length() 호출 안 함
    		System.out.println("문자열 있음");
    }
    ```

    **산술 연산자 우선순위**:

    **비교와 논리 연산자:**

    ```java
    boolean a = true || false && false;  // true || false = true
    boolean b = (true || false) && false; // true && falsse = false
    
    // 비교 연산자는 논리 연산자보다 우선순위 높음
    int x = 5;
    boolean c = x > 3 && x < 10;   // (x > 3) && (x < 10) = true
    ```
    
    **권장: 괄호 사용:**

    ```java
    // 가독성 좋음

    		System.out.println("합격");

    // 복잡한 조건도 괄호로 명확히
    ```

- switch-case 문
    
    **기본 구조:**
    
    ```java
    switch (choice) {
    		case "1":
    				System.out.println("등록을 선택했습니다.");
    				break;
    		case "2":
    				System.out.println("조회를 선택했습니다.");
    				break;
    		case "0":
    				System.out.println("종료합니다.");
    				return;
    		default:
    				System.out.println("잘못된 입력입니다.");
    }
    ```
    
    **case에 사용 가능한 타입:**
    
    | 타입 | Java 버전 | 예시 |
    | --- | --- | --- |
    | `byte`, `short`, `int`, `char` | 1.0+ | `case 1:`, `case 'A':` |
    | `enum` | 5+ | `case MONDAY:` |
    | `String` | 7+ | `case "hello":` |
    
    **사용 불가능한 타입:**
    
    `long`, `float`, `double`, `boolean`, 일반 객체
    
    **break 누락 주의 (Fall-through):**
    
    ```java
    // break이 없으면 다음 case도 실행됨
    int num = 1;
    switch (num) {
    		case 1:
    				System.out.println("1");
    				// break 없음 - fall-through
    		case 2:
    				System.out.println("2");   // num=1이어도 실행됨
    				break;
    		case 3:
    				System.out.println("3");
    				break;
    }
    // 출력: 1, 2
    ```
    
    **Fall-through 활용 (의도적 사용):**
    
    ```java
    int day = 3;  // 수요일
    switch (day) {
    		case 1: case 2: case 3: case 4: case 5:
    				System.out.println("평일입니다.");
    				break;
    		case 6: case 7:
    				System.out.println("주말입니다.");
    		default:
    				System.out.println("잘못된 입력입니다.")
    }
    ```
    
    **Java 14+ 향상된 switch (참고용):**
    
    ```java
    // 화살표 문법 (break 불필요)
    switch (choice) {
    		case "1" -> System.out.println("등록");
    		case "2" -> System.out.println("조회");
    		case "0" -> { return; }
    		default -> System.out.println("잘못된 입력");
    }
    ```
    
    **Switch Expression (Java 14+):**
    
    switch를 문(statement) 이 아닌 식(expression)으로 사용하여 값을 반환할 수 있습니다.
    
    ```java
    // switch expression - 값을 반환
    String message = switch (day) {
        case 1, 2, 3, 4, 5 -> "평일";
        case 6, 7 -> "주말";
        default -> "잘못된 입력";
    };  // 세미콜론 필수!
    
    System.out.println(message);  // "평일" 또는 "주말"
    ```
    
    **yield 키워드 (블록 내에서 값 반환):**
    
    ```java
    String grade = switch (score / 10) {
        case 10, 9 -> "A";
        case 8 -> "B";
        case 7 -> "C";
        default -> {
            System.out.println("재시험 대상");
            yield "F";  // 블록에서는 yield로 값 반환
        }
    };
    ```
    
    **switch 문 vs switch 식 비교**:
    
    | 구분 | switch 문 (statement) | switch 식 (expression) |
    | --- | --- | --- |
    | 값 반환 | 불가능 | 가능 |
    | break/yield | break로 탈출 | yield로 값 반환 |
    | 끝 세미콜론 | 불필요 | **필수** (`;`) |
    | 모든 case 처리 | 선택적 | **필수** (default 포함) |
    
- switch vs if-else 선택 기준
    
    **switch가 적합한 경우:**
    
    - 값이 정확히 일치하는 조건
    - 비교할 값이 3개 이상일 때
    - 코드 가독성이 중요할 때
    
    ```java
    // switch가 적합한 경우
    switch (menuchoice) {
    		case "1": handleAdd(); break;
    		case "2": handleList(); break;
    		case "3": handelEdit(); break;
    		case "4": handleDelete(); break;
    		case "0": exit(); break;
    		default: showError();
    }
    ```
    
    **if-else가 적합한 경우:**
    
    - 범위 조건 ( `x > 10`, `x >= 0 && x < 100`)
    - 복잡한 조건 (`&&`, `||` 조합)
    - null 체크가 필요할 때
    
    ```java
    // if-else가 적합
    if (score >= 90) {
    		grade = "A";
    } else if (score >= 80) {
    		grade = "B";
    } else if (score >= 70) {
    		grade = "C";
    } else {
    		grade = "F";
    }
    ```
    
    **선택 가이드**:
    
    | 상황 | 권장 | 이유 |
    | --- | --- | --- |
    | 메뉴 선택 (1, 2, 3...) | `switch` | 가독성 좋음 |
    | 점수 범위 (90이상, 80이상...) | `if-else` | switch는 범위 불가 |
    | null 가능성 있음 | `if-else` | switch는 null 불가 |
    | 조건이 2개 이하 | `if-else` | switch는 과함 |
    
- 분기문: break, continue, return
    
    **분기문이란?**
    
    - 프로그램의 실행 흐름을 점프시키는 문법
    - 반복문이나 메서드를 제어하는 데 사용
    
    **3가지 분기분 비교:**
    
    | 키워드 | 동작 | 사용 위치 |
    | --- | --- | --- |
    | `break` | 가장 가까운 반복문/switch 탈출 | 반복문, switch |
    | `continue` | 현재 반복을 건너뛰고 다음 반복 | 반복문 |
    | `return` | 메서드 전체 종료 (값 반환) | 메서드 어디서든 |
    
    ---
    
    **break - 반복문/switch 탈출:**
    
    ```java
    // 반복문에서 break
    int i = 0;
    while (i < 10) {
    		i++;
    		if (i == 5) {
    			break;   // 반복문 탈출
    		}
    		System.out.println(i);
    }
    // 출력: 1, 2, 3, 4
    
    // switch에서 break
    switch (choice) {
    		case "1":
    				System.out.println("선택 1");
    				break;   // switch 탈출
    		case "2":
    				System.out.println("선택 2");
    				break;
    }
    ```
    
    ---
    
    **continue - 현재 반복 건너뛰기:**
    
    ```java
    int i = 0;
    while (i < 5) {
    		i++;
    		if (i == 3) {
    		continue;   // 아래 코드 건너뛰고 다음 반복으로
    		}
    		System.out.println(i);
    }
    // 출력: 1, 2, 4, 5 (3은 건너뜀)
    ```
    
    ---
    
    **return - 메서드 종료:**
    
    ```java
    public void processMenu(String choice) {
    		if ("0".equals(choice)) {
    				System.out.println("종료합니다.");
    				return;  // 메서드 종료
    		}
    		
    		System.out.println("선택: " + choice);
    		// return 이후에는 이 코드가 실행됨
    }
    ```
    
    ---
    
    **break vs continue vs return 비교:**
    
    ```java
    public void compare() {
    		int i = 0;
    		while (i < 5) {
    				i++;
    				if (i == 3) {
    						// break;   -> 1, 2 출력 후 반복문 탈출, "끝" 출력
    						// continue;  -> 1, 2, 4, 5 출력 (3만 건너뜀), "끝" 출력
    						// return;    -> 1, 2 출력 후 메서드 종료. "끝" 출력 안 됨
    				
    				}
    				System.out.println(i);
    		}
    		System.out.println("끝");
    }
    ```
    
    ---
    
    **중첩 반복문 break;**
    
    ```java

        int j = 0;

            }
            System.out.println("i=" + i + ", j=" + j);
        }
    }

    ```
